# ReadFlow 产品需求文档

> 版本: v2.3 | 更新: 2026-07-09

---

## 产品定位

**"以书为中心"的阅读记录 App** — 一本书是一个阅读中心，所有阅读行为围绕书统一管理。

## 核心概念

### 📕 Book Hub（一本书 = 一个阅读中心）

每本书包含：基本信息（书名/作者/封面/简介）、阅读状态（待读/阅读中/已完成）、评分、标签、书单归属、阅读时间总计、阅读历史时间线。

### 📖 阅读来源

一本书可关联多种阅读方式：

| 类型 | 说明 | 示例 |
|------|------|------|
| 本地 EPUB/PDF | 导入文件，在 App 内阅读 | 下载的电子书 |
| 纸质书 | 手动计时阅读 | 实体书 |
| 外部平台 | 手动补录时间 | Kindle、微信读书 |

⚠️ **所有来源的阅读时间合并统计到同一本书。**

### ⏱ 阅读计时

| 方式 | 触发 | 场景 |
|------|------|------|
| 自动计时 | 打开电子书 → 开始；退出 → 结束 | 本地 EPUB/PDF |
| 手动计时 | 点击开始 → 点击停止 | 纸质书 |
| 补录时间 | 手动输入时长 | Kindle / 微信读书等 |

### 📥 导入流程

用户可通过以下方式添加书籍：

1. **Discover 搜索添加**：在线搜索书名/作者/ISBN → 找到书籍 → 一键添加到书架
2. **手动创建**：手动输入书名、作者、简介 → 添加到书架
3. **导入本地文件**：在书籍详情页导入 EPUB/PDF/MOBI/TXT 文件作为阅读来源

## 页面清单

### 1. Today（今日阅读概览）— Tab 1
- 时段问候语（早上好/下午好/晚上好）
- 继续阅读英雄卡片（显示当前在读的书 + 进度条）
- 今日阅读统计（今日时长 + 阅读次数）
- 每日阅读目标进度条 + 连续阅读天数
- 最近阅读时间线（最近 4 条）
- 每日名言

### 2. Library（书库）— Tab 2
- 显示所有书本（网格/列表两种视图）
- 筛选：全部 / 阅读中 / 待读 / 已完成（标签 + 书单筛选）
- 搜索：书名/作者
- 在线搜索书籍（书名/作者/ISBN）→ 一键添加到书架
- 书单管理入口
- 长按菜单（编辑/删除/加入书单）
- 导入本地文件 + 手动添加入口
- 空状态引导

### 3. Statistics（阅读统计）— Tab 3
- 总阅读时长、读完本书数量
- 本月/本周阅读时长
- 阅读热力图 + 迷你日历
- 阅读趋势图表（周/月/年维度）
- 时间筛选器（Period Filter）
- 目标管理

### 4. Profile（个人设置）— Tab 4
- 用户信息卡片
- 每日阅读目标设置
- 深色模式切换
- 数据备份 / 恢复 / 导出 / 导入
- 关于信息

### 6. 书籍详情页 — 核心页面
详见 `book/[id].tsx`：
- 封面 + 书名 + 作者 + 简介
- 星级评分 + 状态选择（待读/阅读中/已完成）
- 核心操作：打开阅读 / 开始计时 / 手动记录
- 阅读统计卡片（总时长/次数/最近阅读）
- 阅读来源管理
- 标签 + 书单管理
- 阅读时间线

### 7. 阅读器 — 全屏沉浸
- 简洁阅读界面
- 自动计时
- 字体大小 / 主题切换（亮白/护眼/暗黑）
- 阅读进度保存

### 8. 添加书本 — Modal
- 手动输入书名/作者/简介
- 选择阅读状态
- 保存到书架

### 9. 独立搜索页 — `search.tsx`
- 从 Library 添加菜单进入
- 搜索结果列表（缩略封面 + 书名 + 作者 + 年份）
- 始终显示"手动添加"入口

### 10. 全屏计时器 — `timer.tsx`
- 针对纸质书/外部平台的专注计时
- 页数/章节记录

### 11. 我的阅读 — `my-reading.tsx`
- 从 Statistics 点击条目进入
- 按时间范围（周/月/年）查看阅读详情
- 筛选状态：全部 / 阅读中 / 已完成

### 12. 书单管理 — `collection-manage.tsx`
- 批量管理书单（创建/编辑/删除/排序/隐藏）

### 13. 书单详情页 — `collection/[id].tsx`
- 书单信息卡片 + 操作按钮
- 书内书籍网格浏览 + 空状态

## 非功能需求

- 离线可用（数据存本地 SQLite）
- 安卓优先，同时支持 iOS
- 阅读友好：温暖简约配色，大圆角，充足留白
- 禁止：渐变、毛玻璃、厚重 Material Design 阴影
