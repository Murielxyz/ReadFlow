# ReadFlow 设计规范

> 版本: v2.1 | 更新: 2026-07-04

---

## 设计理念

**温暖 · 简约 · 阅读友好** — 纯白背景，极轻阴影，细线边框，大量留白。像一本干净的精装书，不刺眼，不喧宾夺主。

### 设计规则

- ❌ 禁止渐变、毛玻璃（glassmorphism）
- ❌ 禁止厚重 Material Design 阴影（elevation）
- ✅ 每个卡片最多使用一个强调色
- ✅ 强调色用于小面积元素（Badge、图标、按钮）
- ✅ 大量留白（卡片间距 16px，区块间距 24px）

## 色彩

### 背景色

| Token | 值 | 用途 |
|-------|-----|------|
| `primary` | `#FFFFFF` | 主背景（纯白） |
| `card` | `#F6F6F8` | 卡片背景（极浅灰） |
| `secondary` | `#F7F4EF` | 次级背景 |
| `container` | `#F0EDE8` | 浅灰容器 |
| `pressed` | `#E8E5E0` | 按下态 |

### 文字色

| Token | 值 | 用途 |
|-------|-----|------|
| `primary` | `#1A1512` | 主文字（暖深棕黑） |
| `secondary` | `#8A7A6E` | 次要文字（暖灰棕） |
| `tertiary` | `#A39E99` | 辅助/禁用文字 |
| `inverse` | `#FFFFFF` | 反色（深底白字） |

### 强调色（小面积撞色，5 色轮换）

| 颜色 | 值 | 背景色 | 用途 |
|------|-----|--------|------|
| 🔵 蓝 | `#4A90D9` | `#EEF5FC` | 计时、进度、链接 |
| 🟣 紫 | `#7C6BFF` | `#F4F2FF` | 标签、分类 |
| 🟡 黄 | `#F5A623` | `#FEF8EC` | 评分、收藏 |
| 🟢 绿 | `#50C878` | `#EDF9F2` | 完成、成功 |
| 🩷 粉 | `#FF6B8A` | `#FFF0F3` | 书单、推荐 |

### 状态色

| 状态 | 文字色 | 背景色 |
|------|--------|--------|
| 阅读中 | `#4A90D9` | `#EEF5FC` |
| 待读 | `#A39E99` | `#F0EDE8` |
| 已完成 | `#50C878` | `#EDF9F2` |

### 边框色

| Token | 值 | 用途 |
|-------|-----|------|
| `standard` | `#E5E0DB` | 卡片标准边框（1px） |
| `variant` | `#D5D0CB` | 浅色边框变体 |
| `divider` | `#E5E0DB` | 分隔线 |

## 间距

基于 4px 基准网格：

| Token | 值 | 用途 |
|-------|-----|------|
| `xs` | 4px | 极小间距 |
| `sm` | 8px | 基础间距 |
| `md` | 12px | 中等间距 |
| `lg` | 16px | 卡片内边距、页面边距 |
| `xl` | 24px | 区块间距 |
| `xxl` | 32px | 大区块间距 |
| `xxxl` | 48px | 特大间距 |

- 卡片之间间距：`16px`
- 卡片内边距：`16px`
- 区块之间间距：`24px`
- 屏幕左右边距：`16px`

---

## 圆角

| 级别 | 值 | 适用 |
|------|-----|------|
| `xs` | 4px | 极小元素 |
| `sm` | 8px | 按钮、小卡片 |
| `md` | 12px | 卡片内元素 |
| `lg` | 16px | 普通卡片（默认） |
| `xl` | 24px | 大卡片、封面图 |
| `full` | 9999 | 胶囊按钮、Pill、搜索框 |

---

## 字体

基于 Plus Jakarta Sans：

| 级别 | 大小/字重 | 用途 |
|------|-----------|------|
| `h1` | 24px / Bold (700) | 页面标题、书名 |
| `h2` | 18px / SemiBold (600) | 区块标题 |
| `body` | 14px / Regular (400) | 正文 |
| `caption` | 12px / Medium (500) | 辅助信息、标签 |

---

## 阴影

温暖简约风格使用极轻阴影：

| 类型 | 参数 | 用途 |
|------|------|------|
| `softShadow` | offset(0,1) opacity 0.04 radius 2 | 卡片默认阴影 |
| `softShadowMd` | offset(0,2) opacity 0.06 radius 4 | 浮动元素阴影 |

## 组件规范

### 卡片
- 背景：`#F6F6F8`（极浅灰）
- 圆角：`16px`（`radii.lg`）
- 边框：`1px solid #E5E0DB`
- 阴影：`softShadow`
- 内边距：`16px`

### 按钮
- 主按钮：强调色背景 + 白色文字，圆角 `full`（Pill 形状）
- 次按钮：透明背景 + `#E5E0DB` 边框，圆角 `full`

### Badge / 标签
- Pill 全圆角形状
- 背景为强调色的浅色版本
- 文字为对应强调色

### 搜索框
- Pill 全圆角
- 1px `#E5E0DB` 边框
- 白色背景

## 图标

使用 Ionicons（`@expo/vector-icons`）：
- 今日：`today` / `today-outline`
- 书库：`library` / `library-outline`
- 发现：`search` / `search-outline`
- 统计：`stats-chart` / `stats-chart-outline`
- 我的：`person` / `person-outline`
