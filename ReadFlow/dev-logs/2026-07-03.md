# 2026-07-03 开发日志

## 完成：Phase 15 — 统计增强 + 备份恢复 + 阅读进度

### 15.1 统计趋势图
- **文件**：`app/(tabs)/statistics.tsx`
- 用纯 React Native View 实现 `TrendChart` 柱状图组件，不依赖外部图表库
- 数据从 `useStatistics` heatmap（90天）派生：`weeklyTrend`（近7天）+ `monthlyTrend`（近30天）
- Pill Toggle 切换"本周"/"本月"视图
- 今日柱高亮（全色），历史柱半透明（+50 alpha）
- 自适应缩放（最大高度 120px）、水平滚动（月视图）
- 辅助函数：`getDayLabel()` → "一/二/三/…/日"，`getDateLabel()` → "7/1"

### 15.2 数据备份 + 恢复
- **文件**：`app/(tabs)/profile.tsx`
- 使用 expo-file-system v57 新 API：`Paths.document` + `File` 类（替代旧 `documentDirectory`）
- `File.pickFileAsync()` 替代 expo-document-picker
- **备份**：`new File(Paths.document, fileName).write(json)` → Alert 成功
- **恢复**：`File.pickFileAsync({ mimeTypes: ['application/json'] })` → `result.result.text()` → JSON.parse → INSERT INTO
- 去重策略：书籍按 title+author 匹配；会话/记录/来源用 `INSERT OR IGNORE`
- Profile 数据管理区新增 3 个按钮：导出（Share API）、备份（本地文件）、恢复（选取文件）

### 15.3 Today 页面阅读进度
- **文件**：`src/hooks/useToday.ts`、`app/(tabs)/today.tsx`
- `useToday` Hook 新增字段：
  - `currentBookPageCurrent`：按 `SUM(current_page)` 从 reading_sources 聚合
  - `currentBookPageTotal`：来自 books.page_count
- 英雄卡片改进：显示"第 X/Y 页 · 累计 N 小时"
- 无 page_count 的书籍仅显示时长（向后兼容）

### 关键决策
- 趋势图用纯 RN View，不引入图表库（减少依赖、保持包体积）
- expo-file-system v57 API 是 breaking change（类式 API 替代函数式），无需 expo-document-picker
- 备份/恢复仅限本地文件系统，跨设备同步留待 Phase 16 之后
