**阅读器架构**

```
ReaderScreen
├── WebView (全屏)
│   ├── EPUB: source={{ html: getEpubReaderHTML(fileUri) }}
│   │   └── epub.js CDN 渲染，支持分页、字体缩放、主题切换
│   ├── PDF (iOS): source={{ uri: fileUri }} → Safari 原生渲染
│   ├── PDF (Android): source={{ html: getPdfReaderHTML(fileUri) }}
│   │   └── PDF.js CDN 渲染，canvas 逐页
│   └── 纸质/外部平台: 占位页 + 计时提示
│
├── TopBar (自动隐藏，点击中间切换)
│   ├── ← 返回按钮
│   ├── 章节名 / 来源标签
│   └── 进度百分比
│
├── BottomBar (仅 EPUB/PDF，自动隐藏)
│   ├── ‹ 上一页 / 下一页 ›（Neo-Button 圆形）
│   ├── A⁻ / A⁺ 字体缩放（仅 EPUB，80%-200%）
│   └── 🟠 主题切换（亮白 → 护眼 → 暗黑，3 档循环）
│
└── TapZone (中间透明区，点击切换控制栏)
```

**epub.js 集成细节**

- 使用 jsDelivr CDN 加载 `epubjs@0.3.93`
- RN 调用 `injectJavaScript('readerSetTheme("dark")')` 控制 WebView
- WebView 通过 `window.ReactNativeWebView.postMessage(JSON.stringify({type:'location', percentage, cfi, chapter}))` 回传进度
- 全局命令函数: `readerPrev()`, `readerNext()`, `readerSetFontSize(n)`, `readerSetTheme(t)`, `readerGoTo(pct)`
- 渲染模式: `flow: 'paginated'`（分页），`spread: 'none'`（单页）
- 自动生成 1600 字符粒度的 locations 用于精确跳转

**自动计时逻辑**

- 进入阅读器 → 自动启动计时（标记 `autoStartedTimer`）
- 已在为当前书计时 → 不重复启动
- 另一本书在计时 → 先停止，再为当前书启动
- 离开阅读器 → 如果是自动启动的计时，停止并保存 session

**进度保存**

- EPUB: epub.js `relocated` 事件 → `location.start.percentage` → 0-100
- PDF: 当前页 / 总页数 → 百分比
- 每次位置变化 → 写入 `reading_sources.current_page`
- 返回详情页时 SourceCard 显示最新进度

**UI 交互**

- 控制栏默认可见，点击阅读区中间 → 切换显示/隐藏
- 底部栏 Pill 形状 Neo-Brutalist 风格（硬阴影 + 2px 边框）
