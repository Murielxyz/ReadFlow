## Stitch 设计令牌提取

### 色彩体系

#### 页面背景
| 用途 | Stitch 值 | 说明 |
|------|----------|------|
| 主背景 | `#FAF7F2` | 暖米白色（"纸质感"） |
| Card/Surface | `#FFFFFF` | 纯白卡片 |

#### 品牌色（Primary = 软紫）
| Token | 值 | 用途 |
|-------|-----|------|
| `primary` | `#6a4ea5` / `#A98CE8` | 主按钮、进度条、激活态 |
| `primary-container` | `#a98ce8` | 浅紫容器 |
| `accent-blue-light` | `#E1E0FF` | 淡蓝紫背景 |
| `inverse-primary` | `#d2bcff` | 反色主色 |

#### 辅助色
| Token | 值 | 用途 |
|-------|-----|------|
| `secondary` | `#286a41` / `#8ED2A0` | 绿色 — 完成、成功 |
| `tertiary` | `#7b5900` / `#F6C667` | 金色 — 评分、书签 |
| `error` | `#ba1a1a` | 错误红 |
| `error-container` | `#ffdad6` | 浅红背景 |

#### 粉彩强调色（4 色体系）
| 颜色 | 浅色背景 | 主色 | 用途 |
|------|---------|------|------|
| 🔵 蓝紫 | `#E1E0FF` | `#8B7CF6` | 阅读中、科幻分类 |

#### 文字色

### 阴影

Stitch 设计使用 **硬阴影（Neo-Brutalist 风格）**：
```css
/* 标准卡片阴影 */
box-shadow: 4px 4px 0px 0px #1F1F1F;

/* 小阴影 */
box-shadow: 2px 2px 0px 0px #1F1F1F;

/* Tab Bar */
box-shadow: 1px 1px 0px 0px rgba(31,31,31,1);
```

## 🔴 与 design-spec.md 的关键冲突

### 冲突 1: 页面背景色 ⚠️ 中等
| | Stitch | 现有 Spec |
|--|--------|----------|
| 主背景 | `#FAF7F2` (暖米白) | `#FFFFFF` (纯白) |
| 视觉效果 | 温暖纸质感 | 纯白极简 |

### 冲突 2: 卡片边框色 🔴 严重
| | Stitch | 现有 Spec |
|--|--------|----------|
| 边框色 | `#1F1F1F` (近黑) | `#E5E0DB` (浅灰棕) |
| 视觉效果 | **高对比度、编辑感** | 柔和、低调 |

### 冲突 3: 阴影风格 🔴 严重
| | Stitch | 现有 Spec |
|--|--------|----------|
| 类型 | **硬阴影 (Neo-Brutalist)** | 极轻大气阴影 |
| 参数 | `4px 4px 0px #1F1F1F` | `offset(0,1) opacity 0.04` |
| 规范 | - | ❌ 禁止厚重阴影 |

### 冲突 4: 主色系 ⚠️ 中等
| | Stitch | 现有 Spec |
|--|--------|----------|
| Primary | 紫色 `#6a4ea5` | 蓝色 `#4A90D9` |
| 强调体系 | 4 粉彩 (紫/绿/黄/粉) | 5 标准 (蓝/紫/黄/绿/粉) |

### 冲突 5: 主文字色 ⚠️ 轻微
| | Stitch | 现有 Spec |
|--|--------|----------|
| 主文字 | `#1F1F1F` (纯黑调) | `#1A1512` (暖棕黑) |

### 冲突 6: 圆角 ⚠️ 轻微
| | Stitch | 现有 Spec |
|--|--------|----------|
| 卡片 | `12px` | `16px` |
| 大卡片 | `16px` | `24px` |

### 冲突 7: 底部 Tab Bar 风格 🔴 严重
| | Stitch | 现有 Spec |
|--|--------|----------|
| 激活态 | 紫色背景 + 黑边框 + 硬阴影 | 紫色图标 + 小圆点 |

## 设计风格总结

Stitch 的 "ReadFlow UI Design" 定义了一种 **"现代触感粉彩 + 编辑感边框"** 的混合风格：

| 特征 | 描述 |
|------|------|
| 背景 | 暖米白 `#FAF7F2`，如真实纸张 |
| 卡片 | 纯白 `#FFFFFF` + 1px 近黑边框 |
| 阴影 | 硬阴影（Neo-Brutalist 遗产） |
| 色彩 | 软粉彩 4 色（紫/绿/黄/粉） |
| 圆角 | 中等偏小（12px） |
| 字体 | Plus Jakarta Sans，清晰层级 |
| 间距 | 4px 基准，宽松留白 |
| 图标 | Material Symbols Outlined |
